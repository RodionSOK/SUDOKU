# 🧩 Sudoku Optimization

## 🎯 Управление

### **Клавиатура:**
- **`0-9`** - ввод цифр в активную ячейку
- **`↑↓←→`** - навигация по полю
- **`Enter`** - автоматическое решение текущей головоломки
- **`B`** - запуск полного бенчмарка
- **`R`** - очистка всего поля

### **Мышь:**
- **Клик левой кнопкой** - выбор ячейки

### **Сборка и запуск:**
```bash
# Клонирование репозитория
git clone https://gitlab.com/RodionSOK/sudoku_optimization.git
cd sudoku_optimization

# Сборка
make

# Запуск
make run

# Или напрямую
./sudoku
```

### **Дополнительные команды:**
```bash
make clean    # Очистка файлов сборки
make rebuild  # Полная пересборка
```

## 🔬 Бенчмарк

### **Как использовать:**
1. Запустите программу: `./sudoku`
2. Нажмите клавишу **`B`** в окне программы
3. Смотрите результаты в консоли
